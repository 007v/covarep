Contribute!

We believe that the Covarep repository has a great potential benefit to the speech research community and we hope that you will consider contributing your published algorithms to it. Please see below a few instructions for this purpose.

Submission process

  • Discuss the Intellectual Property (IP) issues with the copyright owner of your code and pick up a convenient License which can be linked to the General Public License Version 3 (GPLv3) (see diagram at License Compatibility). We would encourage you to try to obtain the LGPL licence from your institution in order to be more industry friendly.
  • Update your code to minimize redundancy with the functions already present in the last version of Covarep and the Voicebox.
  • Update your code to make it comply with the Coding Convention while using the Function Template
  • Make a HOWTO script file (see the howtos directory in the repository for examples) which exemplify your code.
  • Test your code on Windows, Mac and Linux platforms.
  • Attach at least one already published paper which describe the suggested method in details.
  • Send your submission either through the GitHub website or by sending the code and the paper by mail to one of the maintainer listed below.
  • A maintainer of the Covarep repository will handle your submission and ask for modification if any points above has to be clarified.
